✨ Interactions
  • Cell hover: scale(1.05), brightness(1.1)
  • Cell tap: haptic feedback (Telegram SDK)
  • Gem reveal: glow animation + bounce
  • Mine hit: shake + red flash + game over modal
  • Win cashout: confetti + green pulse
  • Button press: scale(0.97), 100ms
  • All transitions: 150ms ease-out
