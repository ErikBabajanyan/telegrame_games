🎯 Core value proposition: The fastest way to play provably fair crypto games inside Telegram. One tap to launch, TON wallet connects in seconds, games resolve on-chain.

Product Goals

§5. TON Wallet Integration

Supported Wallets

  • Tonkeeper (primary — most popular TON wallet)
  • MyTonWallet (desktop + mobile)
  • OpenMask (browser extension)
  • TON Space (built into Telegram itself, Q4 2024+)

Deposit Architecture

All deposits flow into a smart contract escrow address. The contract records each user's on-chain address and credits their internal balance. Bets are deducted from the internal balance — not sent on-chain per bet, reducing gas costs significantly.

Withdrawal Architecture

Withdrawal requests trigger a backend-signed transaction from the escrow contract to the user's wallet. Minimum withdrawal: 0.5 TON. The contract enforces withdrawal limits and cooldown periods for risk management.

§6. Security & Provably Fair

Provably Fair System

All game outcomes are determined by a combination of a Server Seed (hashed and revealed after the round) and a Client Seed (user-provided or randomly generated). This ensures neither party can manipulate the result.

🔐 Verification: Any player can independently verify any past game result by inputting the server seed, client seed, and nonce into the published open-source verification tool.

Security Measures

  • All API endpoints protected by JWT + Telegram init data signature verification
  • Rate limiting on bets: max 10 bets/second per user
  • Smart contract audited by third party before mainnet launch
  • Multi-sig treasury wallet for large fund management
  • Automatic withdrawal holds on suspicious patterns (velocity checks)
